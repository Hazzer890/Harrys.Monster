Case study

Building web products with AI

HTML / CSS / JS AI coding tools GitHub Pages In build

The problem

I run WARES, a robotics-education supplier, and it needs a storefront. Robotics West needs a site. This portfolio needed a rebuild. Three real products with real requirements, and one person to build them.

The approach

I build them end to end with AI coding tools: requirements, structure, code, review. The tools write most of the code. The judgement about what to build, what to keep, and what meets the bar stays with me, and every page ships only after I have reviewed it the way I would review a colleague's work.

The state of play

This site is live and was built exactly that way. The WARES storefront and the Robotics West site are in build and not yet public. I would rather show honest work in progress than screenshots of something half-real; when they ship, they will be linked here.
